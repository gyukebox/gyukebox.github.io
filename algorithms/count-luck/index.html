Count Luck

문제

전체 문제: https://www.hackerrank.com/challenges/count-luck/problem

요약하자면, A에서 B로 가는 길이 오직 하나일 때, 갈림길이 몇 번 나오는지 찾는 문제이다. 아래의 그림에서 M이 시작점, *이 종점, X는 벽, .는 길이다.

.X.X......X
.X*.X.XXX.X
.XX.X.XM...
......XXXX.

해결 방법

이런 길찾기 류 문제는 DFS(Depth First Search - 깊이 우선 탐색) 나 BFS(Breadth First Search - 너비 우선 탐색) 를 사용하면 간단하게 해결할 수 있다. 하지만 이 문제만 놓고 보면 DFS를 사용하는 것이 더욱 편리하다

왜 DFS일까?

사실 DFS와 BFS 둘 다 그래프의 모든 노드를 찾는다고 가정하면, 시간복잡도는 O(N) 이 된다. (N이 노드의 갯수라고 가정) 하지만, 이 문제에서 DFS가 편리하다고 생각하는 이유는 바로 문제에서 구해야 하는 값에 있다.

DFS는 보통 재귀함수로 구현하는데, 로직을 간단하게 적어보자면 다음과 같다:

점 A에서:
1. 동쪽으로 가는 가짓수를 모두 구한다
2. 서쪽으로 가는 가짓수를 모두 구한다
3. 남쪽으로 가는 가짓수를 모두 구한다
4. 북쪽으로 가는 가짓수를 모두 구한다
5. (1 + 2 + 3 + 4) 를 하면 갈림길 수가 나온다.

(위 로직은 주어진 문제를 해결하는 데 완벽하지 않다. 추가적인 예외 처리가 필요하다) 5번을 주목하자. 재귀함수 몇 번 짜본 사람이면 바로 감이 올 것이다. 우리가 구하고자 하는 갈림길 은 바로 재귀함수가 종료되어 콜 스택에서 빠지는 시점 이다! 이 점을 잘 활용하면 위의 문제를 해결할 수 있다.

해결하기

DFS를 하기 전에 먼저 준비작업을 거치자. 그래프의 방문 여부를 체크하기 위해서 기존 matrix 와 똑같은 dimension 으로 visited 리스트를 만든다. 그 이후, 시작점과 끝점의 좌표를 구해준 다음, 본 게임으로 들어간다.

def find_start_and_end(matrix, row, col):
    start_point = []
    end_point = []
    for i in range(row):
        for j in range(col):
            if matrix[i][j] == '*':
                end_point = [i, j]
            if matrix[i][j] == 'M':
                start_point = [i, j]
    return start_point, end_point

def countLuck(matrix, k):
    row = len(matrix)
    col = len(matrix[0])
    visited = [[False] * col for _ in range(row)]
    start_point, end_point = find_start_and_end(matrix, row, col)
    count, _ = dfs(matrix, start_point, end_point, visited, 0)
    return 'Impressed' if count == k else 'Oops!'

이제 dfs 를 구현하자.

재귀함수를 짤 때 가장 중요한 건 바로 Base Case 이다. 어떤 경우에 재귀함수를 종료해야 할까? 위의 문제에서는 다음과 같다:

  • 종점에 도달했을 때
  • 종점에 도달하지 못하고 막다른 길에 왔을 때

여기서 또 다른 중요한 점은, 최종적으로 종점에 도달했을 때 갈림길 수만 세서 더해야 한다. “두 갈래 길 모두 막다른 길” 인 경우도 존재하기 때문이다!

우선 우리가 필요한 정보인 가로, 세로, 현재 y좌표와 x좌표, 동서남북 좌표를 다 구하고, 현재 위치의 visited 그래프를 True로 바꿔주었다. 동서남북의 결과가 result 배열에 담기게 될 것이다.

def dfs(matrix, current_point, end_point, visited, count):
    result = []
    row = len(matrix)
    col = len(matrix[0])
    y, x = current_point
    east = x + 1
    west = x - 1
    south = y + 1
    north = y - 1

    visited[y][x] = True

    if current_point == end_point:
    	return count, True

이제 실제로 길을 찾게 되는 부분을 구현해보자. 말로는 간단하면서도, 예외처리를 제대로 안해주어서 문제를 풀 때 자꾸 런타임 에러가 났던 부분이다.

방향 순서는 그냥 동서남북으로 정했고, if문을 살펴보면 예외처리와 이동을 동시에 구현했다. 핵심은 다음과 같다:

  • 가려는 좌표가 길이거나 종점인 경우 이동한다
  • 좌표를 벗어나는 경우 가지 않는다
  • 이미 방문한 곳은 가지 않는다
if east < col and matrix[y][east] in '.*' and not visited[y][east]:
    result.append(dfs(matrix, [y, east], end_point, visited, count))
if west >= 0 and matrix[y][west] in '.*' and not visited[y][west]:
    result.append(dfs(matrix, [y, west], end_point, visited, count))
if south < row and matrix[south][x] in '.*' and not visited[south][x]:
    result.append(dfs(matrix, [south, x], end_point, visited, count))
if north >= 0 and matrix[north][x] in '.*' and not visited[north][x]:
    result.append(dfs(matrix, [north, x], end_point, visited, count))

마지막으로, 갈림길을 구하는 부분을 구현해보자. 우리가 처리해야 하는 경우의 수는 다음과 같이 세 가지다:

  • 길이 하나 이상 있었고, 그 중에 정답인 길이 있는 경우
  • 길이 하나 이상이었지만 그 중에 아무 길도 정답이 아닌 경우
  • 길이 하나도 없었던 경우

맨 처음의 경우에만 우리가 갈림길로 인정해야 하므로, 그런 경우에만 (기존 갈림길 수)에서 1을 더해주고, 정답임을 알려주는 True 도 같이 리턴해주자. (이는 재귀함수가 끝나고 함수가 콜 스택에서 빠지면서, 리턴되는 값에서 다시 쓰인다)

if result:
    for value in result:
    	if value[1]:
    		return value[0] + 1, True
    return count, False
else:
    return count, False

따라서, 전체 dfs 코드는 다음과 같다:

def dfs(matrix, current_point, end_point, visited, count):
    result = []
    row = len(matrix)
    col = len(matrix[0])
    y, x = current_point
    east = x + 1
    west = x - 1
    south = y + 1
    north = y - 1

    visited[y][x] = True

    if current_point == end_point:
        return count, True

    if east < col and matrix[y][east] in '.*' and not visited[y][east]:
        result.append(dfs(matrix, [y, east], end_point, visited, count))
    if west >= 0 and matrix[y][west] in '.*' and not visited[y][west]:
        result.append(dfs(matrix, [y, west], end_point, visited, count))
    if south < row and matrix[south][x] in '.*' and not visited[south][x]:
        result.append(dfs(matrix, [south, x], end_point, visited, count))
    if north >= 0 and matrix[north][x] in '.*' and not visited[north][x]:
        result.append(dfs(matrix, [north, x], end_point, visited, count))

    if result:
        for value in result:
    	    if value[1]:
    	        return value[0] + 1, True
        return count, False
    else:
    	return count, False
